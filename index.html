Создание тестов для РОСТ

Программа для создания тестов для Региональной Образовательной Системы Тестирования

Проект на GitHub Скачать Документация

Региональная образовательная система тестирования («РОСТ») позволяет создвавть учебные тесты, при этом Оценка по тесту автоматически заносится в электронный классный журнал и электронный дневник.

Эта программа позволяет быстро и удобно создавать тесты для РОСТ.

Поддерживаются следующие типы ответов:

  • Один вариант ответа
  • Несолько вариантов ответа
  • Прямой ввод
  • Соответствие
  • Сортировка

Текст вопросов можно наполнять различным контентом:

  • расширенное форматирование текста
  • картинки
  • формулы (греческие и латинские символы, многочисленные матаматические действия(LaTeX))

Если у вас есть тесты в текстовом формате, то вы можете в считанные минуты превратить их в электронные
